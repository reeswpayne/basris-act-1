Welcome to Create Your Kris's documentation!



You probably want to get right in, so let's keep it short. Honestly, the fastest way to get started is to check out the existing encounters and tinker with them. The documentation is for reference if you want to know the specifics of everything. Help is provided on the /r/Unitale subreddit, mostly RhenaudTheLukark#4797.

List of sections with new/changed content in CYK:



Basic setup
Explains the folder and file structure of CYK and mods created with it.

Special Variables
Provides a reference for all variables, new and old, usable in every script in CYK, including the new entity scripts.

Text commands
A new command for face sprites was added, and [func] was changed in an important way.

Game events
Some events have been changed, such as OnHit, and some new events have been added.

Misc. Functions
The function State has been changed, and some new functions have been added.

The Player Object
The Player object now only controls the red soul you use in the menus.
The new Entity Object controls Player characters.

The Entity Object
This section details everything you can do with the new Entity objects in CYK.

The Script Object
Script objects are no longer used for Monster scripts.

Wave Scripts
The Arena now has 3 new properties.

The Inventory Object
Created items can now target players or monsters as you please!

Projectile management
Bullets have some extra variable properties, and the function OnHit has been significantly altered.
